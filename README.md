# MatteBlack

this is an ecommerce website build on nextjs and t3 stack it is more faster than v1 of this projects

### live website

https://www.matteblack.tk/

## installation

install node_modules

```sh
npm install
```

prisma migration

```sh
npx prisma db push
```

then

```sh
npx prisma generate
```

run app in dev mode

```sh
npm run dev
```
