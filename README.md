# Ecommerce
this is a ecommerce app example build with nextjs, typescript, trpc, prisma, mongodb.

htt

## installation
 
install node_modules
```sh
npm install
```

prisma migration
```sh
npx prisma db push
```
then

```sh
npx prisma generate
```

run app in dev mode
```sh
npm run dev
```
